## Quick Installation

### Option 1: Direct Python Installation (Development/Small Teams)

```bash
# Clone repository
git clone https://github.com/cj-wang-sowork/atlas.git
cd atlas

# Create virtual environment
python3 -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install ATLAS modules
python3 -c "import sys; sys.path.insert(0, '.'); from learn_system import *"

# Verify installation
python3 learn_system.py
python3 market_router.py
python3 security_checker.py
python3 hermes_adapter.py
python3 market_intelligence.py
```

### Option 2: Package Installation (Recommended for Enterprise)

```bash
# Create project directory
mkdir -p /opt/atlas
cd /opt/atlas

# Clone repository
git clone https://github.com/cj-wang-sowork/atlas.git ./

# Create virtual environment
python3 -m venv /opt/atlas/venv
source /opt/atlas/venv/bin/activate

# Verify all modules load
python3 -c "
from learn_system import MultiMarketLearnManager
from market_router import MarketRouter, MarketConfig
from security_checker import SecurityChecker
from hermes_adapter import HermesAdapter
from market_intelligence import MarketIntelligence
print('All ATLAS modules loaded successfully')
"
```

## Enterprise Deployment

### System Requirements

- Python 3.8 or higher
- 4GB RAM minimum (8GB recommended for large deployments)
- 50GB disk space (for pattern storage and audit logs)
- Linux/macOS preferred (Windows supported via WSL2)

### 1. Production Directory Structure

```
/var/atlas/
├── venv/                    # Virtual environment
├── config/                  # Configuration files
│   ├── markets.json         # Market definitions
│   ├── security.yaml        # Security policies
│   └── learning.yaml        # Learning configuration
├── logs/                    # Audit and application logs
│   ├── audit.log
│   ├── security.log
│   └── system.log
├── data/                    # Persistent data
│   ├── patterns.json        # Learned patterns
│   ├── violations.json      # Security violations
│   └── backups/             # Regular backups
├── scripts/                 # Operational scripts
│   ├── start.sh
│   ├── stop.sh
│   └── backup.sh
└── atlas/                   # Code repository
    ├── learn_system.py
        ├── market_router.py
            ├── security_checker.py
                ├── hermes_adapter.py
                    └── market_intelligence.py
                    ```

                    ### 2. Initialization Script

                    Create `/var/atlas/scripts/init.sh`:

                    ```bash
                    #!/bin/bash
                    set -e

                    ATLAS_ROOT="/var/atlas"
                    LOG_DIR="${ATLAS_ROOT}/logs"
                    DATA_DIR="${ATLAS_ROOT}/data"
                    CONFIG_DIR="${ATLAS_ROOT}/config"

                    echo "ATLAS Enterprise Initialization"
                    echo "==============================="

                    # Create directories
                    mkdir -p ${LOG_DIR}
                    mkdir -p ${DATA_DIR}
                    mkdir -p ${CONFIG_DIR}
                    mkdir -p ${DATA_DIR}/backups

                    # Set permissions
                    chmod 700 ${ATLAS_ROOT}
                    chmod 700 ${DATA_DIR}
                    chmod 755 ${LOG_DIR}

                    # Initialize logs
                    touch ${LOG_DIR}/audit.log
                    touch ${LOG_DIR}/security.log
                    touch ${LOG_DIR}/system.log

                    # Initialize data
                    echo '{"markets": []}' > ${DATA_DIR}/patterns.json
                    echo '{"violations": []}' > ${DATA_DIR}/violations.json

                    echo "ATLAS initialized successfully"
                    echo "Config directory: ${CONFIG_DIR}"
                    echo "Data directory: ${DATA_DIR}"
                    echo "Log directory: ${LOG_DIR}"
                    ```

                    ### 3. Startup Script

                    Create `/var/atlas/scripts/start.sh`:

                    ```bash
                    #!/bin/bash
                    set -e

                    ATLAS_ROOT="/var/atlas"
                    source ${ATLAS_ROOT}/venv/bin/activate

                    cd ${ATLAS_ROOT}/atlas

                    # Start ATLAS services
                    python3 << 'EOF'
                    import logging
                    from learn_system import MultiMarketLearnManager
                    from market_router import MarketRouter
                    from security_checker import SecurityChecker
                    from hermes_adapter import HermesAdapter
                    from market_intelligence import MarketIntelligence

                    # Configure logging
                    logging.basicConfig(
                        level=logging.INFO,
                            format='%(asctime)s - %(name)s - %(levelname)s - %(message)s',
                                handlers=[
                                        logging.FileHandler('/var/atlas/logs/system.log'),
                                                logging.StreamHandler()
                                                    ]
                                                    )

                                                    logger = logging.getLogger('ATLAS')

                                                    try:
                                                        # Initialize ATLAS services
                                                            router = MarketRouter('/var/atlas/config/markets.json')
                                                                checker = SecurityChecker(audit_log_path='/var/atlas/logs/audit.log')
                                                                    adapter = HermesAdapter(market_router=router, security_checker=checker)
                                                                        intelligence = MarketIntelligence()

                                                                                logger.info("ATLAS Services Initialized")
                                                                                    logger.info(f"Markets: {len(router.markets)} registered")

                                                                                        except Exception as e:
                                                                                            logger.error(f"ATLAS Startup Failed: {e}")
                                                                                                raise
                                                                                                EOF
                                                                                                ```

                                                                                                ## Docker Deployment

                                                                                                ### Dockerfile

                                                                                                Create `Dockerfile`:

                                                                                                ```dockerfile
                                                                                                FROM python:3.11-slim

                                                                                                WORKDIR /app

                                                                                                # Copy repository
                                                                                                COPY . /app/

                                                                                                # Create necessary directories
                                                                                                RUN mkdir -p /app/config /app/logs /app/data

                                                                                                # Set environment
                                                                                                ENV PYTHONUNBUFFERED=1
                                                                                                ENV ATLAS_ROOT=/app
                                                                                                ENV PYTHONPATH=/app:$PYTHONPATH

                                                                                                # Run as non-root
                                                                                                RUN useradd -m -u 1000 atlas
                                                                                                RUN chown -R atlas:atlas /app

                                                                                                USER atlas

                                                                                                # Health check
                                                                                                HEALTHCHECK --interval=30s --timeout=10s --start-period=5s --retries=3 \
                                                                                                    CMD python3 -c "from learn_system import *; print('healthy')" || exit 1
                                                                                                    
                                                                                                    CMD ["python3", "-c", "from learn_system import *; from market_router import *; print('ATLAS ready')"]
                                                                                                    ```
                                                                                                    
                                                                                                    ### Docker Compose
                                                                                                    
                                                                                                    Create `docker-compose.yml`:
                                                                                                    
                                                                                                    ```yaml
                                                                                                    version: '3.8'
                                                                                                    
                                                                                                    services:
                                                                                                      atlas:
                                                                                                          build: .
                                                                                                              container_name: atlas-enterprise
                                                                                                                  ports:
                                                                                                                        - "8000:8000"
                                                                                                                            environment:
                                                                                                                                  - ATLAS_ROOT=/app
                                                                                                                                        - PYTHONUNBUFFERED=1
                                                                                                                                            volumes:
                                                                                                                                                  - ./config:/app/config
                                                                                                                                                        - ./data:/app/data
                                                                                                                                                              - ./logs:/app/logs
                                                                                                                                                                  restart: unless-stopped
                                                                                                                                                                      healthcheck:
                                                                                                                                                                            test: ["CMD", "python3", "-c", "from learn_system import *; print('ok')"]
                                                                                                                                                                                  interval: 30s
                                                                                                                                                                                        timeout: 10s
                                                                                                                                                                                              retries: 3
                                                                                                                                                                                              ```
                                                                                                                                                                                              
                                                                                                                                                                                              Deploy with Docker Compose:
                                                                                                                                                                                              
                                                                                                                                                                                              ```bash
                                                                                                                                                                                              docker-compose up -d
                                                                                                                                                                                              docker-compose logs -f atlas
                                                                                                                                                                                              ```
