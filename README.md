# sowork-claw-workspace

SoWork Agents Claw — VM Workspace

Workspace directory for the SoWork AI platform VM agent (OpenClaw).

## Structure

- `skills/` — Agent skills
- `memory/` — Agent memory files
- `scripts/` — Automation scripts
- `outputs/` — Generated outputs

## Notes

Sensitive files (images, PDFs, credentials) are excluded via .gitignore.
